# ATD3.5-S3-HandySense-Arduino-Library

Arduino library of ATD3.5-S3 + Farm1 shield using HandySense platform for agritech solutions

## Hardware
 * ATD3.5-S3 - esp32 display
 * ATD3.5-S3 Farm1 shield - ATD3.5-S3 extension for agritech solutions
 * Sensor - Your can select sensor for use to automation work and record sensor log

This library support Arduino IDE & PlatformIO
